#### linux命令基础格式

```c++
command [-options] [parameter]

#### 相对路径和绝对路径

**绝对路径：**以根目录作为起点，以/开头

**相对路径：**以当前目录为起点

**特殊路径符**：

```c++
.表示当前目录 ./Desktop 和Desktop意思是一样的
..表示上一级目录
~表示用户的home目录 等价/home/creep/
```





#### ls

**查看目标目录的内容**

**选项**

- -a 显示隐藏的内容（以.开头的文件）
- -l以列表的形式显示内容
- -h 要和-l配合使用，更合适的显示文件的大小单位

**参数**

路径

#### **rm**

**删除文件和文件夹**

**参数**

可以有无穷个，表示要删除的文件和文件夹

**选项**

- -r 删除文件夹
-  -f 强制删除(root用户删除的时候如果不加-f删除的时候要询问你是否删除)

rm命令支持通配符*，用于模糊匹配（即匹配任意内容包含空）

- test*匹配以test为前缀的内容
- *test匹配以test为后缀的内容
- $*test*$ 匹配任何包含test的内容

#### find

**查找指定的文件**

- 按文件名查找

```c++
find 起始路径 -name “被查找的文件名”//文件名支持通配符
```

- 按文件大小查找

```c++
find 起始路径 -size +|-n[kMG]
```

`+`,`-`表示大于小于

n表示大小数字

kMG表示大小单位（k小写其他两个大写）k表示kb，M表示NB，G表示GB

- 查找<10kb的文件

  ```c++
  find / -size -10k
  ```

- 查找>100MB的文件

  ```c++
  find / -size +100M
  ```



#### grep

**从文件中通过关键字过滤文件行**

```c++
grep -[n] 关键字 文件路径
```

- -n加了匹配的行会显示行号
- 关键字表示使用这个关键字来过滤
- 文件路径表示要过滤文件的文件路径 ，也可以用管道符作为输入



#### wc

**统计文件的行数，单词数量，字节数，字符数**

```c++
wc [-c -m -l -w] 文件路径
```

- 不带选项默认统计：行数，单词数，字节数
- -c字节数，-m字符数，-l行数，-w单词数
- 文件路径也可以用管道作为输入



#### 管道符

将管道符左边的结果作为作为右边的输入

```c++
cat 1.txt|grep "111"
```





#### echo

在命令行中输出制定内容

 只有一个参数表示要输出的内容，输出的内容默认有个回车

加了-n选项就没有回车



####  反引号` 

假如直接

```c++
echo pwd 
```

会直接输出pwd，但是此时想把pwd当作指令来使用 ，此时要用反引号将pwd括起来

```c++
echo `pwd`
```

#### tail

**查看文件的尾部内容**

```c++
tail [-f -num] linux路径
```

- -f，表示持续跟踪(按ctrl+c退出)，假如
- -num，表示查看尾部多少行，默认10行
